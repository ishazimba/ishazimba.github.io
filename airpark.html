AIRPARK

Never Worry About Parking Again

Project Overview

The AirPark mobile app offers a seamless and convenient parking experience for travelers, event attendees, and explorers. It enables users to effortlessly find and book parking spaces, while also providing an opportunity for hosts to list their unused spaces and earn extra income.

With AirPark, travelers can easily search for available parking spaces and book them directly through the app. The app provides detailed listings and a map view, making it convenient to find parking in unfamiliar areas. On the other hand, hosts can list their unused parking spaces on the app, monetizing their assets and contributing to the parking ecosystem. Hosts have complete control over managing their listings, including accepting or declining booking requests.

Whether you're planning a trip or attending an event, AirPark simplifies the parking process by offering a user-friendly interface and a wide range of available parking options. By combining innovative features with a seamless user experience, AirPark aims to revolutionize the way people find and book parking spaces.

Key Features

Search and Booking: Travelers can easily search for available parking spaces and book them directly through the app. They can explore detailed listings or visualize available spaces on a map, making it convenient to find parking in unfamiliar areas.


Host Listings: Users can become hosts by listing their unused parking spaces on the app. This allows them to monetize their assets and contribute to the parking ecosystem. Hosts have full control over managing their listings, including accepting or declining booking requests.


Booking Management: Hosts can monitor and manage their listings through a dedicated "Manage Listings" section. They can review and respond to booking requests, ensuring a seamless booking process. Travelers can track the status of their booking requests on the "Manage Parking" page.

Process

This ambitious project was developed collaboratively by a team of five skilled individuals. We utilized React Native Expo to create a native cross-platform mobile application, ensuring compatibility across iOS and Android devices. TypeScript was chosen as the programming language, providing the benefits of static typing and enhanced code reliability.

Throughout the development process, we embraced agile methodologies and employed version control with Git for seamless collaboration. The project's backend was built using PostgreSQL, Node.js, Express, and Google Cloud Storage, while Passport.js enabled secure authentication. For testing purposes, we utilized Jest, a robust JavaScript testing framework.

Technology Stack

This group project was developed using a full-stack approach, leveraging the following technologies:

Backend:

  • PostgreSQL: The project utilized PostgreSQL as the database management system to store and retrieve data efficiently.
  • Node.js and Express: The backend was built using Node.js and Express, providing a robust and scalable foundation for handling API requests and business logic.
  • Google Cloud Storage: The app integrated Google Cloud Storage for secure and reliable storage of user-uploaded content.
  • Passport.js: Authentication was implemented using Passport.js, ensuring secure user registration and login processes.
  • Jest: Unit testing was conducted using the Jest framework to ensure code quality and reliability.

Frontend:

  • Expo: The project utilized Expo, a powerful toolset, and platform for developing cross-platform mobile applications.
  • React Native: The app was built using React Native, enabling the development of a native mobile experience across iOS and Android devices.
  • TypeScript: TypeScript was chosen as the programming language to enhance code quality, maintainability, and developer productivity.
  • Axios: Axios was used for making API requests, facilitating seamless communication between the frontend and backend.
  • Google Places API: Integration with the Google Places API enhanced the app's functionality by providing accurate and comprehensive location data.
  • Tailwind CSS: Tailwind CSS was employed for efficient and responsive UI design, ensuring a visually appealing user interface.
